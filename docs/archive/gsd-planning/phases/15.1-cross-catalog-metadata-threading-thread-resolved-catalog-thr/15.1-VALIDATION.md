the RED task (Task 1) inside each implementation plan, and the GREEN task (Task 2) makes them
pass within the same plan. The Wave 0 obligation — *catalog-discriminating fakes exist before
implementation* — is therefore satisfied per-plan rather than in a standalone wave. The shared
catalog-discriminating reflector (Plan 01) lands in Wave 1 before any consumer.
