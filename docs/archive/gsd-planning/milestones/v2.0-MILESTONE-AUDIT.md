`DialectStrategy` now exposes `build_sample_query`. `QueryService._build_top_query`, `_build_tablesample_query`, `_build_modulo_query` delegate to the dialect: MSSQL emits `TOP`, Databricks and Generic emit `LIMIT`. The `tests/integration/test_sample_data.py::test_get_sample_data_mcp_tool` formerly-deferred test is unskipped and passing.
