<objective>
Add session cleanup (atexit + SIGTERM) and best-effort shutdown, plus error classification for connection failures.

Purpose: CONN-02 requires that all database connections are cleaned up when the MCP server process exits. This plan registers atexit cleanup, adds a SIGTERM handler so atexit fires on process manager termination, makes disconnect_all best-effort (swallows per-engine errors), and adds error classification for actionable user-facing messages on connection failures.

Output: server.py with atexit/SIGTERM registration, connection.py with best-effort disconnect_all and _classify_db_error, tests for all behaviors.
</objective>
