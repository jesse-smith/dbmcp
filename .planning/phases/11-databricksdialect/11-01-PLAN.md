<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| config -> engine URL | Token from TOML config (potentially env var) embedded in connection URL |
| user tool params -> SQL | catalog/schema/table params will be used in DESCRIBE queries (Plan 02) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Information Disclosure | DatabricksDialect.create_engine | mitigate | Token URL-encoded via quote_plus; SQLAlchemy engine.url uses render_as_string(hide_password=True) in logs. Connection model never stores password (NFR-005). |
| T-11-02 | Information Disclosure | ConnectionManager.connect_with_config | mitigate | Token resolved from env var at connection time only, never persisted. URL constructed in local scope, not logged. |
| T-11-03 | Tampering | create_engine URL construction | accept | Host and http_path come from admin-controlled TOML config, not user input. Low risk of injection in URL authority/path position. |
</threat_model>
