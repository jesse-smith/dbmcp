### WIRING-02 (highest severity)

`src/mcp_server/query_tools.py:91` — `get_sample_data`:
```python
metadata_svc = MetadataService(engine)               # dialect NOT passed
query_svc = QueryService(engine, metadata_service=metadata_svc)
```
For Databricks/generic connections, QueryService auto-infer fails silently; `validate_query` gets `dialect="tsql"` and empty `safe_procedures`. Affects VALID-01, VALID-02 for this one tool path.

**Fix:** `dialect = conn_manager.get_dialect(connection_id)`; `MetadataService(engine, dialect=dialect)`; `QueryService(engine, dialect=dialect, metadata_service=metadata_svc)`.

### WIRING-01 (medium severity)

`src/mcp_server/schema_tools.py:170` inside `connect_database` — only affects the schema_count returned in the connect response for generic URLs; subsequent tool calls use the correctly-dialected path.

**Fix:** `MetadataService(engine, dialect=dialect)` (dialect already in scope).
